牛津九上英语训练舱 是一款面向沪教牛津九年级上册英语词汇学习的离线训练网页应用。
项目集成了词库浏览、闪卡记忆、限时闯关、拼写练习、听音辨义、错题复习和学习记录管理等功能，帮助学生更高效地掌握单词与短语。
本项目支持按Unit 单元、题型、题量、难度偏好进行灵活训练，
并通过本地存储自动保存学习进度、答题表现、收藏内容和错题记录，
真正做到“随学随记、随练随复盘”。界面简洁直观，适合日常预习、课堂巩固与考前复习使用。

EnglishOxford Grade9 English Training Hub 
is an offline learning web app designed for vocabulary practice 
in the Shanghai Oxford Grade9 textbook. It integrates multiple study modes, 
including word bank browsing, flashcards, timed quizzes, spelling practice, listening exercises, wrong-answer review, 
and progress tracking, helping students master words and phrases more efficiently.
The project supports flexible training by unit, question type, question count, 
and difficulty preference. It also uses local storage to automatically save learning progress, quiz performance, favorites, 
and wrong answers, making review more personalized and continuous. With a clean and intuitive interface, 
it is ideal for daily preview, classroom reinforcement, and exam preparation.
